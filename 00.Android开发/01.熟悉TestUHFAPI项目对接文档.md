## 关键文件学习

### 1. **AndroidManifest.xml** ⭐⭐⭐

```xml
<!-- 应用的"身份证" -->
- 声明应用权限（网络、存储、相机等）
- 定义应用组件（Activity、Service等）
- 设置应用图标、名称
- 配置启动页面
```

#### 示例

```xml
<?xml version="1.0" encoding="utf-8"?>
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools">

    <application
        android:allowBackup="true"
        android:dataExtractionRules="@xml/data_extraction_rules"
        android:fullBackupContent="@xml/backup_rules"
        android:icon="@drawable/uhf"
        android:label="@string/app_name"
        android:roundIcon="@drawable/uhf"
        android:supportsRtl="true"
        android:theme="@style/Theme.TestUHFAPI"
        tools:targetApi="31">
        <activity
            android:name=".ReadWriteLockActivity"
            android:exported="false" />
        <activity
            android:name=".SettingActivity"
            android:exported="false" />
        <activity
            android:name=".MainActivity"
            android:exported="true">
            <intent-filter>
                <action android:name="android.intent.action.MAIN" />

                <category android:name="android.intent.category.LAUNCHER" />
            </intent-filter>
        </activity>
    </application>

</manifest>
```

#### 解释与分析

这段XML是Android应用的清单文件，定义了应用的基本信息和组件：

1. 应用配置：设置应用图标、名称、主题等基本信息
2. Activity声明：注册了三个Activity组件（ReadWriteLockActivity、SettingActivity、MainActivity）
3. 启动配置：指定MainActivity为应用启动入口点
4. 安全设置：配置数据备份和提取规则，支持RTL布局

**其中MainActivity设置为可导出且作为LAUNCHER类别，是应用的主界面入口。**



- action 标签

作用：定义Activity能够执行的动作类型

**android.intent.action.MAIN：表示这是应用程序的主要入口点，当用户启动应用时首先调用此Activity**

- category 标签

作用：指定Activity的类别或特性

**android.intent.category.LAUNCHER：表示此Activity应该出现在应用程序启动器中，用户可以通过点击应用图标来启动它**

### 2. **build.gradle (app)** ⭐⭐⭐

```gradle
// 应用的构建配置
- applicationId: 应用包名
- minSdk: 最低支持的 Android 版本
- targetSdk: 目标 Android 版本
- dependencies: 依赖的第三方库
```

### 3. **layout/\*.xml** ⭐⭐

```xml
<!-- 界面布局文件 -->
- activity_main.xml: 主界面布局
- 定义按钮、文本框、列表等UI组件
```

### 4. **Java 代码文件** ⭐⭐⭐

```java
// 业务逻辑代码
- Activity: 界面控制器
- Adapter: 数据适配器
- 工具类: 辅助功能
```
